How it works

Intent in. Diff out.

A tight loop from vague idea to verified change — without the ceremony.

  1. 01

    Install the CLI

    Get the daemon-backed AgenC launcher on your machine in one command.

  2. 02

    Open your repo

    Start AgenC where the code lives. It maps the tree, tests, and conventions.

  3. 03

    Plan, edit, verify

    Describe the outcome. AgenC plans, patches, runs checks, and keeps you in the loop.

  4. 04

    Ship

    Clean diffs, remembered context, ready for the next loop.

Install

Get AgenC on your machine.

Use the public launcher. Release artifacts are built from the committed lockfile in agenc-core.

Quick start 
# install the public launcher
npm install -g @tetsuo-ai/agenc

# run in any repo
agenc
Open install docs GitHub repository
