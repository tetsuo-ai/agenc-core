# AgenC runtime

This private package contains the daemon-backed AgenC CLI runtime. Release
artifacts are built from the repository's committed lockfile and distributed by
the public `@tetsuo-ai/agenc` launcher.

See the project
[README](https://github.com/tetsuo-ai/agenc-core#readme) and
[installation guide](https://github.com/tetsuo-ai/agenc-core/blob/main/docs/install.md)
for supported setup and release procedures.
