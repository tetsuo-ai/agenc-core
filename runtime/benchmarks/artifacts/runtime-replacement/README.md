# Runtime Replacement Benchmark Artifacts

This directory holds the locked JSON baselines for the runtime-replacement
benchmark lane.

Rules:

- Baselines are generated by the benchmark runner, not edited by hand.
- File names match the fixture ids in
  `runtime/benchmarks/v1/runtime-replacement/manifest.json`.
- The first green local capture on the pre-Phase-1 `gut` branch is the
  lock point named by `docs/plan/codex-runtime-replacement.md`.

Capture a baseline:

```bash
AGENC_RUNTIME_BENCHMARKS=1 \
AGENC_RUNTIME_BENCHMARK_CAPTURE=1 \
npx vitest run tests/benchmark-runner.integration.test.ts --config runtime/vitest.config.ts
```

Verify against an existing baseline:

```bash
AGENC_RUNTIME_BENCHMARKS=1 \
AGENC_RUNTIME_BENCHMARK_VERIFY=1 \
npx vitest run tests/benchmark-runner.integration.test.ts --config runtime/vitest.config.ts
```
