# @tetsuo-ai/mobile

Private Expo-based mobile client for AgenC.

This workspace owns the mobile app source in `src/` and the Expo launch
surface for device and simulator work.

Local commands:

```bash
npm --prefix mobile run start
npm --prefix mobile run android
npm --prefix mobile run ios
npm --prefix mobile run web
npm --prefix mobile run typecheck
```

This client is part of the product/runtime stack in `agenc-core`.
