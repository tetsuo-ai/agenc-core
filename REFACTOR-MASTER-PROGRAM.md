# agenc-core Repository Role

This repository is the private kernel repo in the post-refactor AgenC topology.

It exists to hold the engine and internal operational surfaces privately while the public contract layer stays in standalone public repositories.

Repo map:

- Public umbrella: `tetsuo-ai/AgenC`
- Public contracts: `tetsuo-ai/agenc-sdk`, `tetsuo-ai/agenc-protocol`, `tetsuo-ai/agenc-plugin-kit`
- Private engine: `tetsuo-ai/agenc-core`
- Private prover: `tetsuo-ai/agenc-prover`
