┤ LIVE LIMITS (CURRENT CONFIG) ├

at one moment, the system can
have up to 16 helper agents
running at the same time.

in a single "spawn wave"
(one planning step), it can
start up to 8 new helper agents.

to finish, it can use at most
32 helper agents total.

helper agents can only hand work
to more helpers up to 4 layers
deep.

┤ SANDBOX CONTAINERS ├

each agent can create their own
secure containers.

desktop.* consumes a desktop
container slot.

the user sets limits depending
on their system.
