# AgenC 0.9.0

Hardened adaptive swarm orchestration, plus the root fix for resumed
sessions' tool cards.

## Features

### Adaptive swarm orchestration, hardened (#1585)

`/swarm` mode and the daemon-backed multi-agent v2 runtime got a
ground-up hardening pass:

- Every delegated assignment now runs on one fresh turn id with its own
  timeout budget and produces one structured outcome receipt, so a
  parent can review, verify, and integrate delegated work against an
  explicit evidence boundary instead of trusting worker chatter.
- Spawn admission, agent identity, role provenance, and per-agent
  mailboxes are enforced at the control layer; `assign_task`
  eligibility is checked atomically (live, idle, reusable descendant
  with no outstanding assignment).
- New worktree isolation module: delegated work can run on an isolated
  copy of the repo with a defined fork-context boundary.
- Conservative routing guidance: `/swarm on` emits one per-turn routing
  reminder with an audit receipt (`agenc.swarm.route.v1`). Sequential
  stays the default; the recommendation is advisory and never bypasses
  permission, sandbox, capacity, admission, or budget controls.
- `/swarm` / `/swarm status` reports effective and saved mode plus
  active and idle/reusable local-agent counts.
- Full product-truth map in
  [docs/design/swarm-orchestration.md](../design/swarm-orchestration.md).

## Fixes

### Resumed sessions render their tools — the root fix

- 0.8.5's `history_replaced` rescan targeted the wrong delivery path.
  The "Tool use unavailable: spawn_agent" cards on resumed swarm
  sessions come from `tool_input_block_start` / `tool_call_started`
  replay: the renderer suppressed the generic collab rows without
  registering the tool names, so exactly those tools failed to parse.
  Both replay paths now register the name (display metadata only — the
  no-duplicate-row contract is unchanged and pinned by test). Verified
  against the originally reported session rollout, including the
  mid-attach truncation state.

## Upgrading

```sh
agenc update            # standalone installs
npm install -g @tetsuo-ai/agenc@latest   # npm installs
```

Restart the daemon after updating: `agenc daemon restart`.
