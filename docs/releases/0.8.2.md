# AgenC 0.8.2

A feature-and-polish release: Ledger wallet integration in the TUI, swarm
mode, and a rebuilt approval-card experience. No state schema change.

## Features

### Ledger

- New `/ledger` command queries the Ledger wallet through wallet-cli without
  leaving the session.
- A Ledger connection indicator sits at the bottom-right of the TUI. Status
  is read on demand rather than polled, so an idle session no longer wakes
  the device path every 15 seconds.
- `exec_command`/`write_stdin` now carry long timeouts sized for
  hardware-approval flows, so waiting for a physical device confirmation no
  longer times the tool out.

### Swarm mode

- `/swarm on|off` toggles swarm mode, with a status-bar indicator, a
  prominent SWARM badge next to the mode label, and a prompt nudge so agents
  know the mode is active.
- Keep-alive swarm workers now render as idle between turns instead of
  appearing to run forever.

### Approval cards

- Plan-approval and generic tool-approval cards are rebuilt as real pickers
  (Claude/Kimi style): the decision row is always visible, the action picker
  can no longer be clipped off the card, and secondary text on the card wash
  is readable.

## Fixes

### TUI

- Ptyxis is opted out of DEC 2026 synchronized output, fixing the chat
  blank-then-repaint flicker.
- New 9-dot spiral activity spinner and a wider agents rail.

### Tools

- `ImagineVideo`/`ImagineImage` have real tool timeouts instead of the
  default.

### Prompting

- Bypass/yolo-permission agents are told to operate autonomously instead of
  pausing to ask permission they already have.

## Upgrading

```sh
agenc update            # standalone installs
npm install -g @tetsuo-ai/agenc@latest   # npm installs
```

Restart the daemon after updating: `agenc daemon restart`.
