# SDK

`@tetsuo-ai/sdk` is owned by the standalone public repository:

- Repo: `https://github.com/tetsuo-ai/agenc-sdk`
- npm: `@tetsuo-ai/sdk`

This local document exists so AgenC internal docs tooling and search still have
an indexed pointer after the SDK left monorepo-local authority.

Canonical public docs:

- README: `https://github.com/tetsuo-ai/agenc-sdk/blob/main/README.md`
- CHANGELOG: `https://github.com/tetsuo-ai/agenc-sdk/blob/main/CHANGELOG.md`
- API baseline: `https://github.com/tetsuo-ai/agenc-sdk/blob/main/docs/api-baseline/sdk.json`

The private kernel in this repo consumes the released SDK package. SDK source
changes and public release work now belong in `tetsuo-ai/agenc-sdk`.
