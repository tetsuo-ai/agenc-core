# Core Scripts

This directory holds repo-level automation for `agenc-core`.

Main script groups:

- boundary checks
- private-kernel distribution and registry flows
- install and pack smoke
- autonomy, watch, and soak helpers
- proof and verifier setup
- deployment and hardening checks

Use the root repo scripts only for `agenc-core` ownership. Umbrella-root
scripts live in `/home/tetsuo/git/AgenC/scripts`.
