# Localnet Social Tools

This workspace owns the localnet social bootstrap and smoke tooling.

Commands:

```bash
npm run localnet:social:bootstrap -- --help
npm run localnet:social:smoke -- --help
```

Direct workspace commands:

```bash
npm run bootstrap --workspace=@tetsuo-ai/localnet-social-tools -- --help
npm run smoke --workspace=@tetsuo-ai/localnet-social-tools -- --help
```
